```css
/*display: flex 指定 flexbox 在父元素上*/
.wrap{
  display: flex;
}
/* flex 的数值设定宽度*/
section:nth-child(1) {
  flex: 2;
}
section:nth-child(2) {
  flex: 1;
}
section:nth-child(3) {
  flex: 1;
}

/*
使用 order 可以改变元素的顺序 默认值是 0 注意正负
flex-direction 可以设置排列方向
*/
@media screen and (max-width: 450px) {
  section:nth-child(1) {
    order: 2;
  }
  .wrap {
    flex-direction: column;
  }
}
```
